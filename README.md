# RedNet Browser

**Anonymous browser for a free internet**

RedNet is a privacy-focused browser with Tor support (via Orbot). Built for Android 8.1+.

## Features

- **Tor / Proxy support** — route traffic through Tor VPN or custom proxy (HTTP/SOCKS4/SOCKS5)
- **Incognito mode** — no history saved
- **Cookie blocking** — block all cookies
- **JavaScript blocking** — block JS for maximum privacy
- **Image blocking** — block images to save bandwidth
- **Clear data on exit** — auto cleanup
- **AdBlock** — active always
- **Widget support**
- **View page HTML source**

## Requirements

- Android 8.1 or higher
- **Orbot** (for Tor mode) — install from Google Play or F-Droid

## How to use Tor

1. Install Orbot
2. Open RedNet → Settings → Tor / Proxy
3. Tap "RUN TOR VPN"
4. Grant permission when asked
